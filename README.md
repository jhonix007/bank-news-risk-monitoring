# AI-модуль выявления и приоритизации банковских риск-инфоповодов

Учебный проект по разработке ML/NLP-сервиса для анализа банковских новостей.

Цель проекта — подготовить основу AI-модуля, который находит упоминания банков в новостном потоке, определяет наличие риск-сигнала и помогает приоритизировать карточки для специалиста.
